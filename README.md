Install python 3.12

## robotpy
this lib is the basic core stuff
`pip install robotpy`
`python -m robotpy init`
`python -m robotpy sync`

To deploy to robot:
`python -m robotpy deploy`

Api docs
https://robotpy.readthedocs.io/projects/robotpy/en/stable/


## pyfrc
